# Tech Stack Overview

## Frontend
- **Next.js 15.3.4** - React framework with App Router
- **TypeScript** - Type safety and better DX
- **Tailwind CSS v4** - Utility-first CSS with new PostCSS plugin

## AI Integration
- **Anthropic Claude 3 Haiku** - Fast, cost-effective model for meal generation
- **Vercel AI SDK** - Streamlined AI integration

## Image Handling
- **Next/Image** - Optimized image loading
- **Pexels** - Static food images (replaced Unsplash due to reliability)

## Development
- **Turbopack** - Fast bundler for development
- **ESLint** - Code quality
- **PostCSS** - CSS processing for Tailwind v4

## Deployment Ready
- Vercel-optimized configuration
- Environment variable support
- API route handlers
